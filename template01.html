HTML5

HTML5でコーディング作業を開始するための、基本的なソースコードのサンプルです。コピーしてご利用いただけます。


<!DOCTYPE html>
<html lang="ja">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<meta name="format-detection" content="telephone=no">

<title>ページタイトル | サイトタイトル</title>
<meta name="description" content="ページ概要">
<meta property="og:title" content="ページタイトル | サイトタイトル">
<meta property="og:type" content="article">
<meta property="og:image" content="http://www.xxxxxx.com/img/xxxxxx.png">
<meta property="og:url" content="http://www.xxxxxx.com/xxxxxx/">
<meta property="og:site_name" content="サイトタイトル">
<meta property="og:description" content="ページ概要">

<!-- common -->
<link rel="stylesheet" href="/css/common.css">
<script src="//ajax.googleapis.com/ajax/libs/jquery/バージョン/jquery.min.js"></script>
<script src="/js/common.js"></script>

<!-- css -->
<link rel="stylesheet" href="/css/xxxxxx.css">

<!-- js -->
<script src="/js/xxxxxx.js"></script>
</head>

<body>

CSS（common.css）

CSS3でコーディング作業を開始するための、基本的なコードのサンプルです。コピーしてご利用いただけます。


@charset "utf-8";

   共通 - common.css                            */

html, body, div, span,
article, section, nav, header, footer, main, aside,
h1, h2, h3, h4, h5, h6,

     -moz-box-sizing: border-box;

JavaScript

JavaScriptやjQueryを利用する場合は、ユーザビリティやアクセシビリティに配慮し、JavaScriptの利用できない環境下でも情報に不足が出ないように考慮しましょう。

jQuery

読み込むjQueryのバージョンは、使用するプラグインの対応状況により変更してください。
特に理由がなければ、GoogleのCDNを利用してください。


<script src="//ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jquery/2.2.4/jquery.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>　※最新
